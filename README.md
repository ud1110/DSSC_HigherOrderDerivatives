This repo has the important datasets and codes for the manuscript titled **Higher-order Derivatives Method for Extracting Single-diode Model Parameters of Dye-sensitized Solar Cells**. All instructions and overview of the data files and codes is provided here:

**1. RTC.ipynb:** This script has the code to compute the parameters for the RTC France solar cell data (a reference dataset taken from literature)

**2. CdTe.ipynb:** This script has the code to compute the parameters for the CdTe solar cell data (this dataset is also taken from literature)
